# Hover Menu 悬浮标签面板 — 设计文档

**日期**: 2026-05-15
**项目**: 截图工具 (screenshot_tool.py)
**状态**: 已确认

---

## 1. 功能概述

截图后悬浮窗底部弹出标签面板，显示 14 个 BMC 区域标签按钮。用户鼠标点击多选标签，点保存按钮自动命名并保存。点"跳过"回退到键盘命名对话框。全程无需键盘。

---

## 2. 确认的需求决策

| 决策项 | 选择 |
|---|---|
| 面板位置 | 悬浮窗正下方 |
| 与命名对话框关系 | 面板替代对话框；点"跳过"才弹对话框 |
| 选择模式 | 多选 + 确认按钮（两键） |
| 标签按钮 | 14 个 NAMING_RULES alias |
| 功能开关 | 悬浮窗新增 Checkbutton，持久化 |

---

## 3. 交互流程

```
截图 → 悬浮窗弹回前台 → 悬浮窗底部出现标签面板
  ├─ 点击标签（亮绿色高亮，可多选）
  │    └─ 点 [✅ 保存] → & 拼接命名 → 编号后缀 → 保存 → 面板消失
  └─ 点 [⏭ 跳过/自定义] → 面板消失 → 弹出键盘命名对话框

面板 5 秒无操作自动消失
```

---

## 4. 自动翻页配合

若 auto_scroll 开启：截图后先翻页 → 再弹面板。

## 6. 技术实现

| 组件 | 方式 |
|---|---|
| 面板窗口 | `tk.Toplevel`, `overrideredirect(True)`, 紧贴悬浮窗下方 |
| 标签按钮 | `tk.Button`, 14 个循环创建，深色主题 |
| 选中状态 | `_selected_tags` set，点击切换 |
| 自动消失 | `after(5000, ...)`，每次点击重置 |
| 保存按钮 | `_on_tag_save()` — 拼接命名 → 编号 → 保存 PNG |

## 8. 边缘情况

| 场景 | 处理 |
|---|---|
| 关闭开关 | 不弹面板，走键盘命名对话框 |
| 5 秒不操作 | 自动消失 |
| 同名文件 | 编号后缀递增 |
| 面板弹出时悬浮窗被遮挡 | 悬浮窗有置顶，面板跟随 |
| 与键盘补全的共存 | 面板替代对话框，键盘补全在对话框中保留 |
